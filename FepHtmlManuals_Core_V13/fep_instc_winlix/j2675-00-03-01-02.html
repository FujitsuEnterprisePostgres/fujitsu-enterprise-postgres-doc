Note

The following JDKs or JREs are available:

  • JDK or JRE shipped with the Interstage Application Server

  • Oracle JDK or JRE

  • An OpenJDK that has passed the Java TCK (Technology Compatibility Kit)

It is recommended that the JDK or JRE included with Interstage Application Server is used.
