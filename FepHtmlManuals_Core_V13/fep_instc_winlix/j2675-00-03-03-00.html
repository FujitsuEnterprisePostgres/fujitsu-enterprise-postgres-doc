3.3 Uninstallation

This section describes the procedure for uninstalling the Linux client.

Note

  • Before uninstalling the product, close the product program and all applications that are using it.

  • On SLES 12, before uninstallation, ensure that Oracle JRE 8 is installed, and export the JAVA_HOME environment variable.

    #export JAVA_HOME="oracleJre8InstallDir"
