2.1 Required Operating System

One of the operating systems shown below is required in order to use FUJITSU Enterprise Postgres.

  • RHEL7.2 or later minor version

  • RHEL8.1 or later minor version

  • SLES 12 SP5

Note

SLES 12 cannot be used if performing failover operation integrated with PRIMECLUSTER.
