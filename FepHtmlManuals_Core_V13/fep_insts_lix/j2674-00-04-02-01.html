4.2.1 Creating an Instance Administrator

Decide which OS user account will be assigned the instance administrator role. You can assign it to a new user or to an existing one, but you cannot assign it to the OS superuser (root).

The following example shows an OS user account with the name "fsepuser" being assigned the instance administrator role.

Example
# useradd fsepuser# passwd fsepuser

Note

The following note applies if using WebAdmin for operations:

  • If the password is changed for the user account of the instance administrator, set the changed password using ALTER ROLE WITH ENCRYPTED PASSWORD. This will encrypt the password with the default MD5 algorithm. To encrypt the password with the SCRAM-SHA-256 algorithm, set the password encryption parameter to "scram-sha-256" using SET password_encryption = ‘scram-sha-256’ prior to using the ALTER ROLE function.
