D.2 pgx_confidential_policies

This table provides information on masking policies.

Column

Type

Description

schema_name

varchar(63)

Schema name of table for which a masking policy is applied

table_name

varchar(63)

Name of table for which a masking policy is applied

policy_name

varchar(63)

Masking policy name

expression

varchar(1024)

Masking condition

enable

boolean

Masking policy status

  • 't': Enabled

  • 'f': Disabled

policy_description

varchar(1024)

Masking policy description

Execution example

postgres=# select * from pgx_confidential_policies; schema_name | table_name | policy_name | expression | enable | policy_description -------------+------------+-------------+------------+--------+-------------------- public      | t1         | p1          | 1=1        | t      | (1 row)
