C.3 pgx_stat_latch

The pgx_stat_latch view displays statistics related to latches, with each type of wait information within FUJITSU Enterprise Postgres displayed on a separate line.

Table C.2 pgx_stat_latch view

Column

Type

Description

latch_name

name

Name of the latch

total_waits

bigint

Number of waits caused a wait

total_wait_time

double precision

Number of milliseconds spent in waits caused by the latch

stats_reset

timestamp with timezone

Last time at which this statistic was reset
