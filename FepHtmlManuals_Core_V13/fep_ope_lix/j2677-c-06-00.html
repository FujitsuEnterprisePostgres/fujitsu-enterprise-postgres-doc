C.6 pgx_stat_gmc

The pgx_stat_gmc view provides information about the GMC areas.

Table C.5 pgx_stat_gmc view

Column

Type

Description

searches

bigint

Number of times the cache table is searched.

hits

bigint

Number of times the cache table is hit.

size

bigint

The current amount of memory (bytes) used in the GMC area.

stats_reset

timestamp with timezone

Last time these statistics were reset.
